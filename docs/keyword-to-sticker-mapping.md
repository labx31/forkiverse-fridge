When a project is posted with hashtags like `#vibecoding` or `#fridge`, the system analyzes:
1. Post content & title

Then matches keywords in this priority order:
1. **Tech Keyword Overrides** (highest priority - unique assignments)
2. **General Sticker Keywords** (broader matches)
3. **Fallback Stickers** (when nothing matches)
